# HallSync – Faculty Lecture Hall Management & Smart Scheduling System

## Introduction

HallSync is a smart scheduling and lecture hall management system developed for the **Faculty of Computing at Sabaragamuwa University of Sri Lanka**. The system aims to simplify the management of lecture timetables, hall bookings, and academic scheduling through a centralized digital platform.

Currently, many universities rely on **printed timetables or manual coordination** to manage lecture halls and schedules. This often causes confusion, scheduling conflicts, and delays in communication between students, lecturers, and administrative staff. HallSync addresses these issues by providing a **mobile-first solution** that allows users to access updated timetables, check hall availability, and receive real-time notifications about schedule changes.

The system integrates several important features such as:

- Lecture timetable management  
- Hall booking system  
- Real-time notifications  
- Google Maps navigation for lecture halls  

These features enable better coordination within the faculty. By digitizing these processes, the application helps **reduce scheduling conflicts, improve communication, and make academic operations more efficient** for students, lecturers, and administrators.
